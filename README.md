# Tech Buzz — Blogs

A static tech-news and blogging website — "a heaven for technophiles." A front-end project featuring a
news layout, article pages, search, and a feedback/blogging section.

## About

Tech Buzz is a multi-page static site presenting technology news and blog content with a clean,
responsive layout built from HTML and CSS.

## Getting started

No build step is required — it's a static site.

```bash
# Open directly in a browser
open index.html

# ...or serve locally
python3 -m http.server 8000   # then visit http://localhost:8000
```
